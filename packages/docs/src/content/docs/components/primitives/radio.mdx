The Radio component provides single-selection controls built on [Radix UI RadioGroup](https://www.radix-ui.com/primitives/docs/components/radio-group). Use RadioGroup to wrap a set of RadioGroupItem elements, ensuring only one option can be selected at a time.

## Import

```tsx
import { RadioGroup, RadioGroupItem } from '@tiinno-ui/components';
```

## Sizes

RadioGroup and RadioGroupItem do not have built-in size variants. Sizing can be controlled through className on the individual items.

### Survey Question

```tsx
function SurveyQuestion({ question, options, value, onChange }) {
  return (
    <div className="space-y-3">
      <p className="text-sm font-medium">{question}</p>
      <RadioGroup value={value} onValueChange={onChange}>
        {options.map((option) => (
          <div key={option.value} className="flex items-center space-x-2">
            <RadioGroupItem value={option.value} id={`q-${option.value}`} />
            <label htmlFor={`q-${option.value}`} className="text-sm">
              {option.label}
            </label>
          </div>
        ))}
      </RadioGroup>
    </div>
  );
}
```

## Accessibility

The RadioGroup component is built on Radix UI and follows WAI-ARIA best practices:

- Uses `role="radiogroup"` on the container
- Each item uses `role="radio"` with proper `aria-checked` state
- Supports roving tabindex for keyboard navigation within the group
- Focus visible states for keyboard users
- Disabled state properly communicated to assistive technology

### Keyboard Support

| Key | Action |
|-----|--------|
| Tab | Moves focus to the radio group (or the selected item) |
| Arrow Down / Arrow Right | Moves focus and selection to the next item |
| Arrow Up / Arrow Left | Moves focus and selection to the previous item |
| Space | Selects the focused item (if not already selected) |

### Best Practices

- Always pair RadioGroupItem elements with visible labels using `id` and `htmlFor`
- Wrap radio groups in a `fieldset` with a descriptive `legend` for form context
- Use RadioGroup for mutually exclusive choices; use Checkbox for multi-select
- Provide a reasonable default selection when appropriate
