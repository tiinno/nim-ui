The Separator component renders a thin line to visually divide content sections. Built on [Radix UI Separator](https://www.radix-ui.com/primitives/docs/components/separator), it supports horizontal and vertical orientations with proper accessibility semantics.

## Import

```tsx
import { Separator } from '@nim-ui/components';
```

## Basic Usage

<ComponentPreview title="Horizontal Separator">

## Accessibility

- When `decorative={true}` (default): renders with `role="none"`, ignored by screen readers
- When `decorative={false}`: renders with `role="separator"`, announced to screen readers
- Proper `aria-orientation` attribute for vertical separators
