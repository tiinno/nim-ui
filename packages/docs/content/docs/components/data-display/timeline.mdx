import {
  Timeline,
  TimelineContent,
  TimelineDescription,
  TimelineItem,
  TimelineMarker,
  TimelineTime,
  TimelineTitle,
} from '@/components/nim';

The Timeline component presents audit trails, order activity, and status history as a compact semantic list.

## Import

```tsx
import {
  Timeline,
  TimelineContent,
  TimelineDescription,
  TimelineItem,
  TimelineMarker,
  TimelineTime,
  TimelineTitle,
} from '@nim-ui/components';
```

## Backoffice Guidance

- Use `Timeline` for audit trails, not long-form narratives.
- Keep titles short and put extra context in `TimelineDescription`.
- Use `TimelineTime` with `dateTime` so machines and assistive tools get the exact timestamp.
